Comando dos Git

  • git init - Criar novo repositório.
  • git status - Verificar estado dos arquivos/diretórios.
  • git add - Adicionar arquivo/diretório.
  • git commit - Comitar arquivo/diretório.
  • git rm - Remover arquivo/diretório.
  • git log - Visualizar histórico.
  • git remote / git remote -v - Exibir os repositórios remotos
  • git remote add origin - Vincular repositório local com um repositório remoto.
  • git remote show origin - Exibir informações dos repositórios remotos.
  • git remote rename origin - Renomear um repositório remoto.
  • git remote rm - Desvincular um repositório remoto.
  • git clone - Clonar um repositório remoto já existente.
  • git bisect - Realiza um bisect.
  • git stash - Cria um stash.
  • git branch - Cria uma branch.
  • git tag - Cria uma tag.
  • git pull / git fetch - Atualizar repositório local de acordo com o repositório remoto.
